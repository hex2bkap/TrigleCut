# TrigleCut 操作ガイド

画像ファイルを縦・横・中間（ほぼ正方形）の3種類に自動で振り分けるツールです。

---

## 基本的な使い方

### ステップ1：フォルダを選択する

振り分けたい画像が入っているフォルダを指定します。

- **「フォルダを選択...」ボタン**を押してフォルダ選択ダイアログから選ぶ
- または、エクスプローラーからフォルダをドラッグして枠内に**ドロップ**する

---

### ステップ2：設定を確認・変更する

| 項目 | 内容 |
|---|---|
| **出力先** | 振り分け先フォルダをどこに作るか。「ソースフォルダ内に作成」が標準。別の場所に出力したい場合は「別のフォルダを指定」を選ぶ |
| **操作** | 元ファイルをコピーするか移動するかを選ぶ。コピーが安全 |
| **重複ファイル** | 同名ファイルが出力先にすでにある場合の動作。スキップ（何もしない）が標準 |
| **判定閾値** | 縦横比（長辺 ÷ 短辺）がこの値未満の画像を「中間」として扱う。1.0 が完全な正方形。**初期値 1.1 はほとんどの場合そのまま使える。**「中間に入るものが多すぎる」と感じたら小さく、「縦/横のつもりの画像が中間に入る」と感じたら大きくする |
| **サブフォルダ** | チェックを入れると、選択フォルダの中にあるフォルダも再帰的に処理する |
| **フォルダ名** | 作成される振り分け先フォルダの名前。自由に変更できる |

---

### ステップ3：スキャンする

**「スキャン実行」ボタン**を押します。

- 選択したフォルダ内の画像を読み込み、縦・横・中間の枚数を集計します
- 処理中はステータスバーに進行状況が表示されます
- 完了すると「縦 ○枚・横 ○枚・中間 ○枚」のように結果が表示されます

---

### ステップ4：振り分けを実行する

スキャン完了後、**「振り分け実行」ボタン**を押します。

- 出力先に「縦」「横」「中間」フォルダが自動で作成され、ファイルが振り分けられます
- 処理中はプログレスバーで進捗が確認できます
- **「キャンセル」ボタン**を押すといつでも中断できます（処理済みのファイルはそのまま残ります）
- 完了するとステータスバーに結果が表示されます

---

## 判定のしくみ

| 判定結果 | 条件 |
|---|---|
| **縦** | 長辺が高さ方向（縦長画像）、かつ縦横比が閾値以上 |
| **横** | 長辺が幅方向（横長画像）、かつ縦横比が閾値以上 |
| **中間** | 縦横比が閾値未満（ほぼ正方形な画像） |

縦横比の計算式：`長辺 ÷ 短辺`（例：1000×800px の画像は 1.25）

スマートフォンで撮影した写真は、ファイル内の回転情報（EXIF）を考慮して正しく判定します。

---

## 対応フォーマット

JPG / JPEG / PNG / WEBP / BMP / GIF / TIFF / TIF / HEIC / HEIF

> HEIC・HEIF の読み込みには Windows の「HEIFイメージ拡張機能」コーデックが必要です（Microsoft Store から無料でインストールできます）。

---

## 注意事項

- **「移動」モードを使う場合は注意**。元のフォルダからファイルが移動されます。不安な場合はまずコピーで試してください
- **「サブフォルダも処理する」**を使う場合、すでに作成済みの「縦」「横」「中間」フォルダの中は処理対象から自動で除外されます
- スキップされたファイルの一覧はエラーログに記録されます（`%LocalAppData%\TrigleCut\logs\`）
